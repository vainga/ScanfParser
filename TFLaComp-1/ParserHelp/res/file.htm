Реализованные функции

  2. Открыть
  3. Сохранить
  4. Сохранить как
  5. Выход

Создать

Создает новый пустой файл для программы и очищает содержимое поля для ввода.

Открыть

Открывает файл и выводит текст программы в поле для ввода.

Сохранить

Сохраняет текст в текущем открытом файле или новом файле, если ранее файл не был создан.

Сохранить как

Открывает окно ввода имени файла и выбора директории для сохранения.

Выход

Закрывает приложение. Если есть несохраненные изменения, сообщает об этом.
