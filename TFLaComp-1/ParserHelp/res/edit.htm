Реализованные функции

  1. Отменить
  2. Повторить
  3. Вырезать
  4. Копировать
  5. Вставить
  6. Удалить
  7. Выделить все

Отменить

Отменяет последнее действие в поле ввода текста.

Повторить

Повторяет отмененное действие в поле ввода текста.

Вырезать

Удаляет текст из поля ввода и сохраняет его в буфере обмена.

Копировать

Копирует текст из поля ввода в буфере обмена.

Вставить

Вставляет текст из буфера обмена в поле ввода.

Удалить

Удаляет текст из поля ввода.

Выделить все

Выделяет весь текст в поле ввода.
